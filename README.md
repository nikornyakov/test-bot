# 🏀 Lets Play Basketball Bot

Telegram бот для автоматизации управления баскетбольной командой. Бот автоматически рассылает опросы о посещении тренировок, напоминания и информационные сообщения.

## 📋 Описание

Этот проект представляет собой Telegram бота, который помогает управлять расписанием тренировок баскетбольной команды. Бот автоматически:

- Отправляет опросы по понедельникам и средам о посещении предстоящих тренировок
- Рассылает напоминания по вторникам и четвергам
- Отправляет приветственные сообщения с актуальным расписанием
- Ведет подробное логирование всех действий

## ✨ Возможности

### Автоматические сообщения
- **Опросы посещаемости**: По понедельникам и средам бот отправляет опросы о посещении тренировок
- **Напоминания**: По вторникам и четвергам бот отправляет напоминания о предстоящих тренировках
- **Приветственные сообщения**: Информационные сообщения с расписанием и важными новостями

### Ручное управление
- Возможность отправки приветственного сообщения вручную
- Тестовые опросы для проверки работоспособности

### Безопасность и надежность
- Использование переменных окружения для хранения чувствительных данных
- Подробное логирование всех операций
- Graceful обработка ошибок

## 🛠 Требования

## 📦 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/lets-play-basketball.git
   cd lets-play-basketball
   ```

2. **Создайте виртуальное окружение:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   # или
   venv\Scripts\activate     # Для Windows
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте конфигурацию:**
   ```bash
   cp config.json config.local.json  # Для локальных настроек
   # Отредактируйте config.local.json по необходимости
   ```

## ⚙️ Настройка

### Переменные окружения

Создайте файл `.env` в корне проекта:

```bash
# Токен вашего Telegram бота (получить у @BotFather)
BOT_TOKEN=your_bot_token_here

# ID группы, куда будут отправляться сообщения
GROUP_ID=-1001234567890
```

### Конфигурация

Проект использует JSON-конфигурацию для настройки сообщений, расписания и других параметров. Основной файл конфигурации `config.json` содержит:

- **schedule**: Расписание тренировок и опросов
- **venue**: Информация о месте проведения
- **messages**: Шаблоны всех сообщений бота
- **team**: Информация о команде
- **logging**: Настройки логирования

### Как получить BOT_TOKEN

1. Напишите [@BotFather](https://t.me/botfather) в Telegram
2. Используйте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте токен из ответа BotFather

### Как получить GROUP_ID

Запустите скрипт для определения ID группы:

```bash
python get_group_id.py
```

Или используйте GitHub Action "Get Group ID 2" в разделе Actions репозитория.

## 🏗 Архитектура

Проект построен по модульной архитектуре с разделением ответственности:

### Основные модули

- **`bot_base.py`**: Базовый класс `TelegramBotBase` с общей функциональностью для всех ботов
- **`bot_config.py`**: Класс `Config` для работы с JSON-конфигурацией
- **`simple_bot.py`**: Основная логика бота с функциями отправки сообщений
- **`get_group_id.py`**: Утилита для получения ID групп
- **`test_poll.py`**: Тестовый скрипт для проверки работоспособности

### Конфигурационная система

- **`config.json`**: Основной файл конфигурации
- **`.env`**: Переменные окружения для чувствительных данных
- **`.env.example`**: Шаблон переменных окружения

### Система тестирования

- **`tests/`**: Директория с тестами
- **`pytest.ini`**: Конфигурация pytest
- Полное покрытие unit и integration тестами

### Особенности архитектуры

- **Наследование**: Все компоненты наследуются от базовых классов
- **Конфигурируемость**: Все тексты и настройки вынесены в конфигурацию
- **Тестируемость**: Полная система тестирования с моками
- **Расширяемость**: Легкое добавление новых функций через конфигурацию

### Локальный запуск

#### Основной бот
```bash
python simple_bot.py
```

Бот автоматически определит текущий день недели и выполнит соответствующее действие:
- **Понедельник/Среда**: Отправка опроса о посещении тренировки
- **Вторник/Четверг**: Отправка напоминания
- **Другие дни**: Ничего не делает

#### Ручная отправка приветственного сообщения
```bash
python simple_bot.py welcome
```

#### Тестирование опросов
```bash
python test_poll.py
```

## 🔧 Разработка

### Добавление новых функций

1. Создайте новую функцию в `simple_bot.py`
2. Добавьте соответствующую логику в `main()`
3. Обновите документацию

### Логирование

Все действия бота логируются в файлы с расширением `.log`. Логи включают:
- Время выполнения
- Уровень логирования
- Подробное описание действий
- Информацию об ошибках

### Обработка ошибок

Бот включает comprehensive обработку ошибок:
- Проверка наличия переменных окружения
- Валидация входных данных
- Graceful handling сетевых ошибок Telegram API
