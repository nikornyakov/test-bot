# 🎯 Простая система сбора данных для отчетов

## 📋 Что это?

Простая и надежная система для сбора опросов из Telegram группы и генерации отчетов с реальными данными.

## 🚀 Быстрый старт

### Шаг 1: Настройка бота

1. **Добавьте бота** `@tests_nvk_bot` в вашу группу
2. **Сделайте бота администратором**
3. **Отправьте** `/start` в группе

### Шаг 2: Настройте переменные

Создайте `.env` файл:
```bash
BOT_TOKEN=8301930933:AAGxtEK99cyhv1h6e6XUZPOWZyPFDHWE8FY
GROUP_ID=-1001234567890  # РЕАЛЬНЫЙ ID вашей группы
```

### Шаг 3: Соберите данные

```bash
# Собрать опросы из группы
python3 simple_poll_collector.py
```

### Шаг 4: Сгенерируйте отчет

```bash
# Годовой отчет
python3 reports/generate_reports.py yearly 2025

# Или просто просмотреть
python3 simple_analytics.py
```

## 📊 Как это работает?

### 1. Сбор данных (`simple_poll_collector.py`)

- Подключается к Telegram Bot API
- Получает обновления из группы
- Находит сообщения с опросами
- Сохраняет в `reports/polls.json`

**Формат данных:**
```json
{
  "poll_123": {
    "poll_id": "poll_123",
    "date": "2025-03-12",
    "question": "Тренировка 12.03.2025",
    "total_votes": 15,
    "votes": {
      "✅ Буду": 12,
      "❌ Не смогу": 3
    }
  }
}
```

### 2. Аналитика (`simple_analytics.py`)

- Загружает данные из `polls.json`
- Рассчитывает статистику посещаемости
- Формирует красивый отчет

### 3. Отправка (`generate_reports.py`)

- Использует аналитику для создания отчета
- Отправляет в Telegram группу
- Сохраняет данные в репозиторий

## 🚨 Возможные проблемы

### "Обновления не получены"
**Решение:**
- Бот добавлен в группу?
- Отправлено сообщение в группе?
- GROUP_ID правильный?

### "Опросы не найдены"
**Решение:**
- Создайте новый опрос в группе
- Проверьте права бота
- Запустите сборщик снова

### "Ошибка отправки"
**Решение:**
- Проверьте BOT_TOKEN
- Убедитесь что бот в группе
- Проверьте интернет соединение

## 🔄 Автоматизация

### GitHub Actions
Добавьте в `.github/workflows/analytics.yml`:
```yaml
- name: Collect polls
  run: python3 simple_poll_collector.py

- name: Generate report
  run: python3 reports/generate_reports.py yearly 2025
```

### Локальный запуск
```bash
# Ежедневный сбор
python3 simple_poll_collector.py

# Еженедельный отчет
python3 reports/generate_reports.py weekly
```
