# Настройка опросов для тренировок на улице

## Обзор

Дополнительный функционал для отправки опросов о тренировках на улице по воскресеньям в 13:00 с прогнозом погоды от Яндекс.Погоды.

## Как это работает

- **Автоматический запуск**: Каждую субботу в 10:00 МСК
- **Ручной запуск**: Через GitHub Actions
- **Цель**: Собрать информацию о количестве участников для воскресной тренировки
- **Фишка**: Автоматическое добавление прогноза погоды для СПб на завтра

## Настройка секретов

Добавьте в репозиторий GitHub следующие Secrets:

1. `OUTDOOR_BOT_TOKEN` - токен бота для уличных тренировок
2. `OUTDOOR_GROUP_ID` - ID чата для уличных тренировок
3. `YANDEX_WEATHER_API_KEY` - API ключ Яндекс.Погоды

### Как получить API ключ Яндекс.Погоды

1. Перейдите на [https://developer.tech.yandex.ru/](https://developer.tech/yandex.ru/)
2. Войдите в аккаунт Яндекса
3. Создайте новое приложение
4. Подключите API "Погода"
5. Скопируйте API ключ

### Можно использовать того же бота

Если хотите использовать того же бота:
- `OUTDOOR_BOT_TOKEN` = `BOT_TOKEN`
- `OUTDOOR_GROUP_ID` = ID нового чата

## Локальный запуск

```bash
# С экспортом API ключа
export YANDEX_WEATHER_API_KEY="ваш_ключ"
python simple_bot.py outdoor
```

## Автоматизация

- **Суббота 10:00 МСК** - автоматический опрос о воскресной тренировке с прогнозом погоды
- **Ручной запуск** - в любой момент через GitHub Actions

## Текст опроса

```
Баскетбол на улице в воскресенье (дата) 🏀

Варианты ответа:
✅ Буду
❌ Не смогу  
🤔 Еще не знаю
⏰ Опоздаю
```

## Прогноз погоды

Бот автоматически добавляет в сообщение:
- 🌡️ Температуру
- ☁️ Состояние погоды
- 💨 Скорость ветра
- 🌧️ Количество осадков
- 🧥/👕/☔ Рекомендации по одежде

## Дополнительная информация

Опрос включает напоминание о том, что нужно взять:
- Удобную одежду для улицы
- Кроссовки для уличной игры
- Воду
- Защиту от солнца (по погоде)

## Обработка ошибок

Если API погоды недоступен, бот отправит сообщение: "🌤️ Прогноз погоды временно недоступен"
